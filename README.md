# projet_angular

# ** Front-end: **

You must have npm and the Angular CLI installed.

Run npm install

Run ng serve for a dev server. Navigate to http://localhost:4200/.

# ** Back-end: **

Run npm install

Install nodemon 

Run nodmeon serve

# ** Packages to install: **

sequelize: npm install sequelize

bcrypt: npm install bcrypt

Crypto-js: npm install crypto-js
